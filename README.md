# Heart Disease Prediction using Machine Learning

## Project Overview
This project focuses on predicting heart disease using machine learning techniques based on clinical data. Two models were developed and compared to evaluate their performance.

---

## Objectives
- Predict whether a patient is at risk of heart disease
- Compare performance between different machine learning models
- Evaluate models using multiple metrics

---

## Dataset
The dataset contains clinical features such as:
- Age
- Cholesterol
- Blood Pressure
- Maximum Heart Rate
- Other medical attributes

---

## Methods

### 1. Data Preprocessing
- Handling missing values
- Feature selection
- Data splitting into training and testing sets
- Data scaling (for Logistic Regression)

### 2. Models Used
- Logistic Regression
- Random Forest Classifier

---

## Model Evaluation

The models were evaluated using:

- Accuracy Score
- Confusion Matrix
- Classification Report
- ROC Curve
- AUC Score

---

## Results

- Both models performed well on the dataset
- Random Forest showed slightly better performance based on AUC score
- ROC curve comparison was used to visualize performance differences

---

## Feature Importance
Random Forest was used to identify the most important features contributing to heart disease prediction.

---

## Prediction
The trained model can take new patient data and predict whether the patient is likely to have heart disease.

---

## Conclusion
In this project, two machine learning models were developed to predict heart disease using clinical data. The models were evaluated using multiple performance metrics, and Random Forest demonstrated slightly better performance. This highlights the potential of machine learning in assisting early diagnosis and clinical decision-making.
